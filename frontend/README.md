# Affiliate Marketing System - Frontend

Next.js frontend dashboard for the affiliate marketing system.

## Features

- Real-time analytics dashboard
- Product management
- Campaign tracking
- Landing page management
- Email subscriber management
- Conversion tracking

## Environment Variables

- `NEXT_PUBLIC_API_URL` - Backend API URL (https://affiliate-backend-production-2a1c.up.railway.app/api)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This project is configured to deploy to Vercel automatically when changes are pushed to the main branch.
